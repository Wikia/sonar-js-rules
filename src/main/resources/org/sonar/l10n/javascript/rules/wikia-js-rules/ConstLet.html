const and let variable declarations are an ES6 feature. They are not part of the ES5 specification, and are unsupported by many browsers. A normal var declaration should be used instead.

Noncompliant Code Example

	const CONST = 3.14;

	function foo() {
		let a = 5;
	}

Compliant Solution

	var CONST = 3.14;

	function foo() {
		var b = 5;
	}
