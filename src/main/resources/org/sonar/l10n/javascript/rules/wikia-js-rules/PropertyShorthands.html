Object property shorthands and variable keys are an ES6 feature. They are not part of the ES5 specification, and are unsupported by many browsers. Pair properties with strong literal or identifier keys should be used instead.

Noncompliant Code Example

	let o = {
		x, y
		z() {},
	};

Compliant Solution

	var o = {
		x: x,
		y: y,
		z: function() {}
	};
