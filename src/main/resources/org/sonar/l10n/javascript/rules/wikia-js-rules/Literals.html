Binary and octal numeric literals are an ES6 feature. They are not part of the ES5 specification, and are unsupported by many browsers. A decimal or hexadecimal numeric literal should be used instead.

Noncompliant Code Example

	let
		a = 0b0110,
		b = 0o01;

Compliant Solution

	var
		a = 0x5,
		b = 1;
