Iterators are an ES6 feature. They are not part of the ES5 specification, and are unsupported by many browsers. A for-in loop should be used to iterate over objects, while a for/while loop or Array.forEach should be used to iterate over arrays.

Noncompliant Code Example

	let a = [];

	for (let item of a) {

	}

Compliant Solution

	var a = [];

	a.forEach(function(item) {

	});
