Class declarations are an ES6 feature. They are not part of the ES5 specification, and are unsupported by many browsers. A function or an object literal should be used instead.

Noncompliant Code Example

	class Clazz {
		constructor() {
			this.bar = 5;
		}
		foo() {}
	}

Compliant Solution

	function Clazz() {
		this.bar = 5;
	}

	Clazz.prototype.foo = function() {};
